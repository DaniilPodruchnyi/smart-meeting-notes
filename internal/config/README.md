# `internal/config`

## Назначение

Централизованная загрузка настроек из `.env` и переменных окружения.

## Что реализовать

- Адрес HTTP-сервера (если используется), уровень логов (уже есть базово).
- `TELEGRAM_BOT_TOKEN` и при webhook — базовый URL / секрет пути.
- Параметры PostgreSQL: `DATABASE_URL` или отдельно host, port, user, password, dbname, sslmode.
- Учётные данные и базовые URL для **SaluteSpeech** и **GigaChat** (токены, scope — по документации Sber).
- Таймауты и лимиты (например, максимальный размер файла, интервал polling SaluteSpeech).

## Правила

- Не импортировать сюда `telebot`, HTTP-клиенты и SQL — только чтение окружения и валидация значений.
