# `internal/logger`

## Назначение

Единообразное логирование на базе `go.uber.org/zap`.

## Что реализовать

- Настройка уровня из `config` (уже есть базово).
- При необходимости — дополнительные поля по умолчанию (например, `service` name).
- Обертки-хелперы под типичные операции (опционально): `WithRequestID`, контекстный логгер.

## Правила

- Usecase и адаптеры получают `*zap.Logger` (или интерфейс логгера) снаружи, не создают глобальный логгер сами.
