# `internal/adapters/telegram`

## Назначение

Интеграция с Telegram: библиотека `gopkg.in/telebot.v3`, обработчики входящих обновлений, скачивание файлов по `FileURL`.

## Что реализовать по ТЗ

- `OnText`: разбор команд `/start`, `/list`, `/get`, `/find`, `/chat` и аргументов; делегирование в usecase; отправка ответа пользователю.
- `OnVoice` / `OnAudio`: получить ссылку на файл, скачать байты (или stream), передать в usecase / очередь на транскрипцию.
- Поддержка **нескольких пользователей** одновременно (бот stateless на уровне обработчиков).
- Модуль «клиента Telegram» в этом пакете: обёртка над тем, что нужно для загрузки файла по URL из Bot API.

## Связь с очередью

- Долгие операции (SaluteSpeech, GigaChat) не блокировать навсегда поток обновлений: ставить задачу в канал, затем по готовности слать ответ в **тот же** chat/user id.

## Правила

- Парсинг команд и форматирование сообщений пользователю — здесь или в тонком слое; бизнес-правила — в usecase.
