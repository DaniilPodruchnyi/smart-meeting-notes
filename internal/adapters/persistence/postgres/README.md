# `internal/adapters/persistence/postgres`

## Назначение

Реализация репозиториев на PostgreSQL (подключение из `cmd/server`, передача в usecase).

## Что реализовать

- `Open` / пул соединений из DSN из `config`.
- Репозитории:
  - пользователи: upsert при `/start`;
  - встречи: insert после успешной транскрипции, update при добавлении саммари;
  - поиск: запросы под `/find`;
  - список и get по id с фильтром по `telegram_user_id`.
- Транзакции, если одна операция трогает несколько таблиц.
- При желании — папка `migrations/` SQL-файлами или инструмент миграций.

## Правила

- Не вызывать Telegram и GigaChat из этого пакета.
