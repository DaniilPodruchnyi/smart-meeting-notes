# `cmd/server`

## Назначение

Единая точка входа приложения.

## Что реализовать здесь

- Инициализация `config` (путь к `.env`, при необходимости флаги CLI).
- Создание `zap`-логгера.
- Подключение к PostgreSQL (pool / одно соединение — по договорённости).
- Сборка репозиториев и клиентов (`SaluteSpeech`, `GigaChat`).
- Создание usecase-слоя с внедрёнными зависимостями.
- Запуск **либо** long-polling Telegram-бота, **либо** HTTP-сервера для webhook (если выберете webhook) — плюс graceful shutdown для обоих.
- Запуск воркера/пула воркеров очереди (каналы), чтобы обработка API не блокировала приём обновлений от Telegram.

## Не размещать здесь

Бизнес-логику и HTTP-клиенты к внешним API — только wiring и `main`.
