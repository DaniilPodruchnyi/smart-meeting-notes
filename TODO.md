## Карта проекта: папки и ответственность

Ориентир по [docs/STRUCTURE.md](docs/STRUCTURE.md). Пересечения (схема БД, интерфейсы репозиториев) — договаривайтесь в **Shared**.

| Путь | Основной ответственный | Что там (кратко) |
|------|------------------------|------------------|
| `cmd/server` | **Shared** (сборка) | `main`: config, логер, БД, клиенты, бот + shutdown |
| `internal/config` | **Shared** | переменные `.env`, DSN, токены API |
| `internal/logger` | **Shared** | zap |
| `internal/domain` | **Shared** | сущности без SDK (User, Meeting, Transcript…) |
| `internal/app/usecase` | **B** (логика), **A** (вызовы из бота) | сценарии: транскрипция→саммари→БД, поиск, `/chat` |
| `internal/adapters/telegram` | **A** | `telebot`, `OnText` / `OnVoice` / `OnAudio`, скачивание файлов |
| `internal/adapters/salutespeech` | **A** | async REST: upload → task → poll → текст |
| `internal/adapters/gigachat` | **B** | токен, саммари, ответы для `/chat` |
| `internal/adapters/persistence/postgres` | **B** (репозитории, поиск), **A** может помочь с insert после STT | PostgreSQL, `/list`, `/get`, `/find` |
| `internal/queue` | **A** (часто в паре с telegram) | каналы, воркеры, маршрутизация ответов по `user/chat` id |
| `internal/server` и `…/transport/http` | **Shared** | HTTP, `/ping`, health, при необходимости webhook |
